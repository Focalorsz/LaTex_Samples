# 📄 LaTeX Samples & Editorial Templates

Bem-vindo ao repositório **LaTeX_Samples**.

Este projeto reúne modelos e exemplos desenvolvidos para demonstrar competências em **editoração científica**, **diagramação acadêmica**, **documentação técnica** e **estruturação de documentos complexos utilizando LaTeX**.

O objetivo é disponibilizar um conjunto de templates reutilizáveis que evidenciam boas práticas de organização, padronização e automação de documentos — habilidades frequentemente utilizadas em ambientes acadêmicos, editoriais e institucionais.

---

## 🎯 Objetivos do Repositório

- Demonstrar domínio prático de LaTeX e Overleaf;
- Apresentar modelos reutilizáveis para diferentes contextos acadêmicos;
- Exibir conhecimento em editoração científica e documentação técnica;
- Servir como portfólio de projetos relacionados à produção e organização de conteúdo acadêmico;
- Explorar recursos avançados como BibTeX, referências cruzadas, tabelas complexas e formatação em múltiplas colunas.

# 🚀 Como Utilizar

## Overleaf

1. Baixe a pasta do projeto desejado;
2. Acesse o Overleaf;
3. Clique em **New Project → Upload Project**;
4. Envie o projeto compactado;
5. Compile normalmente.
